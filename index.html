Design System

A white-label CSS design system with 300+ tokens, 19 components, multi-theme support, and WCAG 2.2 accessibility.

Quick Start

Include the stylesheet in your HTML to get every component, token, and theme:

<link rel="stylesheet" href="css/design-system.css">

Dark Mode

Enable dark mode by setting the data-mode attribute on the root element:

<html data-mode="dark">

Light mode is the default. All semantic tokens automatically remap when the mode changes.

Theming

Switch brand colors with the data-theme attribute. Three themes are included:

<!-- Default theme (Blue) — no attribute needed -->
<html>

<!-- Ocean theme (Cyan) -->
<html data-theme="ocean">

<!-- Forest theme (Emerald) -->
<html data-theme="forest">

Themes and modes combine freely: data-theme="ocean" data-mode="dark" gives you Ocean in dark mode.
