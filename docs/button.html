Button

Buttons trigger actions. Available in 6 variants and 3 sizes, with support for icons, loading, and disabled states.

ds-btn__iconSVG icon inside button

Accessibility

Accessibility Guidelines

Use <button> for actions and <a> for navigation. Icon-only buttons must include an aria-label that describes the action. The focus ring is always visible on keyboard focus via :focus-visible. Disabled buttons use the native disabled attribute and are excluded from tab order automatically.
