Checkbox

Checkboxes allow users to select one or more options from a set. Built with native <input type="checkbox"> for full accessibility.

Default

<label class="ds-checkbox-label">
  <input type="checkbox" class="ds-checkbox">
  Unchecked
</label>

<label class="ds-checkbox-label">
  <input type="checkbox" class="ds-checkbox" checked>
  Checked
</label>

Indeterminate

<label class="ds-checkbox-label">
  <input type="checkbox" class="ds-checkbox" id="select-all">
  Select all
</label>

<script>
  // Indeterminate must be set via JavaScript
  document.getElementById('select-all').indeterminate = true;
</script>

Sizes

<label class="ds-checkbox-label">

  Small (16px)
</label>

  Medium (20px, default)

Accessibility

Accessibility Guidelines

Uses native <input type="checkbox"> for built-in keyboard and screen reader support. Wrap related checkboxes in a <fieldset> with a <legend> to provide group context to assistive technology. The indeterminate state must be set via JavaScript. Focus ring is visible on keyboard navigation via :focus-visible.
