Checkbox

Checkboxes allow users to select one or more options from a set. Built with native <input type="checkbox"> for full accessibility.

When to use

Use checkboxes when
Zero, one, or multiple options can be selected from a set. Or for a single binary choice with a visible label: "I agree to the terms", "Remember me".
Don't use checkboxes when
Only one option can be selected from a group (use Radio), a quick on/off toggle with immediate effect is needed (use Toggle), or when displaying a long list of options (use Select).
ComponentSelectionContext
CheckboxMultiple selections allowedForm context, explicit submit
ToggleSingle on/offImmediate effect, no submit needed
RadioMutually exclusive, one must be selectedForm context, explicit submit

Anatomy

Option label 1 2 3 4
1 Label (.ds-checkbox-label) — wraps checkbox + text, extends click target.
2 Input (.ds-checkbox) — styled native <input type="checkbox">.
3 Checkmark — CSS pseudo-element, visible when checked.
4 Indeterminate dash — CSS pseudo-element, visible when indeterminate = true.

Best practices

Interests
Do
Wrap in <fieldset> with <legend> for screen reader context.
Don't
Use checkboxes without labels. Every checkbox needs a visible text label.
Do
Use indeterminate for "select all" when some children are checked.
Don't
Use indeterminate as a third user-facing state. It is a programmatic state only.

Content guidelines

RuleExample
Use affirmative statements"Receive marketing emails" — not "Don't send me emails"
Group legend should be a question or instruction"Select your interests" — not "Interests list"
Keep labels concise"Remember me" — not "Keep me logged in on this device for future sessions"
Use sentence case"Receive notifications" — not "Receive Notifications"

Implementation notes
Uses native <input type="checkbox"> for built-in keyboard and screen reader support. The indeterminate state must be set via JavaScript (el.indeterminate = true). Always wrap related checkboxes in a <fieldset> with a <legend> to provide group context to assistive technology.
