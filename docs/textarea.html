Textarea

Textareas let users enter multi-line text. Uses a wrapper pattern similar to the Input component.

Default

<div class="ds-textarea">
  <textarea class="ds-textarea__field" placeholder="Enter your message..."></textarea>
</div>

Sizes

<div class="ds-textarea ds-textarea--sm">
  <textarea class="ds-textarea__field" placeholder="Small"></textarea>
</div>

<div class="ds-textarea ds-textarea--md">
  <textarea class="ds-textarea__field" placeholder="Medium"></textarea>
</div>

<div class="ds-textarea ds-textarea--lg">
  <textarea class="ds-textarea__field" placeholder="Large"></textarea>
</div>

Character Counter

0 / 200 
<div class="ds-field">
  <label class="ds-field__label" for="bio">Bio</label>
  <div class="ds-textarea">
    <textarea class="ds-textarea__field" id="bio"
      data-maxlength="200" data-counter="bio-counter"></textarea>
  </div>
  <span class="ds-field__counter" id="bio-counter">0 / 200</span>
</div>

States

Error

Message must be at least 20 characters. 
<div class="ds-field ds-field--error">
  <label class="ds-field__label" for="msg">Message</label>
  <div class="ds-textarea">
    <textarea class="ds-textarea__field" id="msg"
      aria-invalid="true" aria-describedby="msg-error"></textarea>
  </div>
  <span class="ds-field__error" id="msg-error">Message must be at least 20 characters.</span>
</div>

Disabled

<div class="ds-textarea">
  <textarea class="ds-textarea__field" disabled>Disabled textarea</textarea>
</div>

Readonly

<div class="ds-textarea">
  <textarea class="ds-textarea__field" readonly>Read-only content</textarea>
</div>

CSS Classes

ClassDescription
ds-textareaWrapper element for the textarea
ds-textarea__fieldThe native <textarea> element
ds-textarea--smSmall size
ds-textarea--mdMedium size (default)
ds-textarea--lgLarge size
ds-textarea--errorError state with red border
ds-textarea--disabledDisabled state (or use native disabled)
ds-textarea--readonlyReadonly state (or use native readonly)
ds-field__counterCharacter counter element
ds-field__counter--overApplied when character count exceeds max

Accessibility

Accessibility Guidelines

Always pair textareas with a visible <label>. Use aria-invalid="true" and aria-describedby to associate error messages with the field. The character counter should use aria-live="polite" for dynamic updates so screen readers announce remaining characters.
