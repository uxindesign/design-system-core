Layout de aplicaçãoApplication layout
O dashboard usa display: grid com 2 colunas (sidebar 320px + conteúdo) e 2 linhas (topbar 48px + conteúdo). A estrutura de grade e as dimensões usam exclusivamente tokens var(--ds-dimension-*) e var(--ds-size-*). The dashboard uses display: grid with 2 columns (320px sidebar + content) and 2 rows (48px topbar + content). Grid structure and dimensions use only var(--ds-dimension-*) and var(--ds-size-*) tokens.
