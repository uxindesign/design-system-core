Foundation ColorsFoundation Colors

Paleta de cores primitivas. Valores brutos sem significado semântico — os tokens semânticos (Theme) referenciam esta coleção via alias. Ordem espelha a collection Foundation no Figma. Cada swatch mostra o nome completo da CSS variable e o valor hex/rgba como tags de token, e indica contraste WCAG 2.2 contra White e Black: AAA ≥ 7:1, AA ≥ 4.5:1, AA Large ≥ 3:1, Fail < 3:1.Primitive color palette. Raw values with no semantic meaning — Theme tokens alias this collection. Order mirrors the Foundation collection in Figma. Each swatch shows the full CSS variable name and hex/rgba value as token tags, and indicates WCAG 2.2 contrast against White and Black: AAA ≥ 7:1, AA ≥ 4.5:1, AA Large ≥ 3:1, Fail < 3:1.
