Relacao com tokens de overlayRelationship with overlay tokens

Tokens de opacidade controlam a propriedade CSS opacity de um elemento. Tokens de overlay (--ds-color-overlay-black-*, --ds-color-overlay-white-*) usam valores rgba() e sao aplicados como background-color. Servem propositos diferentes:Opacity tokens control the opacity CSS property on an element. Overlay tokens (--ds-color-overlay-black-*, --ds-color-overlay-white-*) use rgba() values and are applied as background-color. They serve different purposes:
