Card

Cards group related content and actions. Available in 3 variants: default, outlined, and elevated.

Default

Default Card

This is the default card style with a subtle border. Use it for general content grouping throughout the interface.

Outlined

Outlined Card

The outlined variant uses a visible border for clear separation. Ideal for forms, settings panels, or content that needs defined boundaries.

Elevated

Elevated Card

The elevated variant uses a box shadow instead of a border. Use it for content that should appear lifted above the surface, such as modals or popovers.
