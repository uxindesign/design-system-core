Reduced MotionReduced motion

prefers-reduced-motion
Usuários que habilitam movimento reduzido nas configuracoes do SO terao todas as transições e animações reduzidas a quase zero. Isto e aplicado globalmente no reset base -- componentes individuais não precisam tratar isso.Users who enable reduced motion in their OS settings will have all transitions and animations shortened to near-zero. This is enforced globally in the base reset -- individual components do not need to handle it.
@media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; scroll-behavior: auto !important; } }

Referencia WCAG: 2.3.3 Animation from Interactions (AAA). Embora o DS tenha como alvo AA, isto e implementado porque o custo e quase zero e o impacto para usuários sensiveis a movimento e significativo.WCAG reference: 2.3.3 Animation from Interactions (AAA). While the DS targets AA, this is implemented because the cost is near-zero and the impact for motion-sensitive users is significant.
