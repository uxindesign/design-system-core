TematizaçãoTheming

3 temas × 2 modos = 6 combinações visuais. Temas alteram as cores da marca; modos alternam light/dark.3 themes × 2 modes = 6 visual combinations. Themes change brand colors; modes switch light/dark.

Como funcionaHow It Works

O design system usa dois atributos HTML no elemento raiz para controlar a aparência:The design system uses two HTML attributes on the root element to control appearance:

  • data-theme controla a paleta da marca (cores primária e secundária).data-theme controls the brand palette (primary and secondary colors).
  • data-mode alterna entre tokens de superficie/texto light e dark.data-mode switches between light and dark surface/text tokens.

Ambos os atributos são independentes e podem ser combinados livremente. Todos os componentes respondem automaticamente às mudanças porque referenciam tokens semânticos, não valores hardcoded.Both attributes are independent and can be combined freely. All components automatically respond to changes because they reference semantic tokens, not hardcoded values.

Criando um tema personalizadoCreating a Custom Theme

Crie seu próprio tema adicionando um novo seletor data-theme que sobrescreve os tokens semânticos primário e secundário. Siga este padrão:Create your own theme by adding a new data-theme selector that overrides the primary and secondary semantic tokens. Follow this pattern: 

/* Custom "Sunset" theme */
[data-theme="sunset"] {
  --ds-brand-default:        #EA580C; /* Orange 600 */
  --ds-brand-hover:  #C2410C; /* Orange 700 */
  --ds-brand-active: #9A3412; /* Orange 800 */
  --ds-brand-subtle: #FED7AA; /* Orange 200 */
  --ds-state-hover:  #FFF7ED; /* Orange 50  */
  --ds-brand-content-default:   #C2410C; /* Orange 700 */

  --ds-accent-default:        #DB2777; /* Pink 600 */
  --ds-accent-hover:  #BE185D; /* Pink 700 */
  --ds-accent-active: #9D174D; /* Pink 800 */
  --ds-accent-subtle: #FBCFE8; /* Pink 200 */
  --ds-accent-subtle:  #FDF2F8; /* Pink 50  */
  --ds-accent-content-default:   #BE185D; /* Pink 700 */

  --ds-content-link-default:       #EA580C;
  --ds-content-link-hover: #C2410C;
  --ds-focus-ring: 0 0 0 2px var(--ds-surface-default),
                   0 0 0 4px #EA580C;
}

/* Optional: dark-mode adjustments for your theme */
[data-theme="sunset"][data-mode="dark"] {
  --ds-brand-default:        #FB923C; /* Orange 400 */
  --ds-brand-hover:  #FDBA74; /* Orange 300 */
  --ds-brand-active: #FED7AA; /* Orange 200 */
  --ds-brand-subtle: #7C2D12; /* Orange 900 */
  --ds-state-hover:  #7C2D12;
  --ds-brand-content-default:   #FB923C;

  --ds-content-link-default:       #FB923C;
  --ds-content-link-hover: #FDBA74;
  --ds-focus-ring: 0 0 0 2px var(--ds-surface-default),
                   0 0 0 4px #FB923C;
}

Depois aplique com <html data-theme="sunset">. Todo componente que usa --ds-brand-default adotará sua nova paleta automaticamente.Then apply it with <html data-theme="sunset">. Every component that uses --ds-brand-default will pick up your new palette automatically.
