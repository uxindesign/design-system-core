Theming

3 themes × 2 modes = 6 visual combinations. Themes change brand colors; modes switch light/dark.

How It Works

The design system uses two HTML attributes on the root element to control appearance:

  • data-theme controls the brand palette (primary and secondary colors).
  • data-mode switches between light and dark surface/text tokens.

Both attributes are independent and can be combined freely. All components automatically respond to changes because they reference semantic tokens, not hardcoded values.

Available Themes

Default (Blue)

No data-theme attribute needed. Primary: Blue 600, Secondary: Purple 600.

<html>
  <!-- Default theme, no attribute required -->
</html>

Ocean (Cyan)

Primary: Cyan 600, Secondary: Indigo 600.

<html data-theme="ocean">

Forest (Emerald)

Primary: Emerald 600, Secondary: Amber 600.

<html data-theme="forest">

Dark Mode

Add data-mode="dark" to the root element. All surface, background, text, and border tokens are remapped automatically. Light is the default. 

<!-- Light mode (default) -->
<html>

<!-- Dark mode -->
<html data-mode="dark">

All 6 Combinations

Every theme works with both modes. Here are all six combinations showing a primary button in each:

<html>
<html data-mode="dark">
<html data-theme="ocean">
<html data-theme="ocean" data-mode="dark">
<html data-theme="forest">
<html data-theme="forest" data-mode="dark">

Creating a Custom Theme

Create your own theme by adding a new data-theme selector that overrides the primary and secondary semantic tokens. Follow this pattern: 

/* Custom "Sunset" theme */
[data-theme="sunset"] {
  --ds-color-primary:        #EA580C; /* Orange 600 */
  --ds-color-primary-hover:  #C2410C; /* Orange 700 */
  --ds-color-primary-active: #9A3412; /* Orange 800 */
  --ds-color-primary-subtle: #FED7AA; /* Orange 200 */
  --ds-color-primary-muted:  #FFF7ED; /* Orange 50  */
  --ds-color-primary-text:   #C2410C; /* Orange 700 */

  --ds-color-secondary:        #DB2777; /* Pink 600 */
  --ds-color-secondary-hover:  #BE185D; /* Pink 700 */
  --ds-color-secondary-active: #9D174D; /* Pink 800 */
  --ds-color-secondary-subtle: #FBCFE8; /* Pink 200 */
  --ds-color-secondary-muted:  #FDF2F8; /* Pink 50  */
  --ds-color-secondary-text:   #BE185D; /* Pink 700 */

  --ds-text-link:       #EA580C;
  --ds-text-link-hover: #C2410C;
  --ds-focus-ring: 0 0 0 2px var(--ds-surface-default),
                   0 0 0 4px #EA580C;
}

/* Optional: dark-mode adjustments for your theme */
[data-theme="sunset"][data-mode="dark"] {
  --ds-color-primary:        #FB923C; /* Orange 400 */
  --ds-color-primary-hover:  #FDBA74; /* Orange 300 */
  --ds-color-primary-active: #FED7AA; /* Orange 200 */
  --ds-color-primary-subtle: #7C2D12; /* Orange 900 */
  --ds-color-primary-muted:  #7C2D12;
  --ds-color-primary-text:   #FB923C;

  --ds-text-link:       #FB923C;
  --ds-text-link-hover: #FDBA74;
  --ds-focus-ring: 0 0 0 2px var(--ds-surface-default),
                   0 0 0 4px #FB923C;
}

Then apply it with <html data-theme="sunset">. Every component that uses --ds-color-primary will pick up your new palette automatically.
