ElevationElevation

Escala oficial de elevação com reset técnico. Em light mode, sombras criam profundidade. Em dark mode, superfícies progressivamente mais claras indicam elevação -- sombras são secundárias.Official elevation scale with a technical reset. In light mode, shadows create depth. In dark mode, progressively lighter surfaces indicate elevation -- shadows are secondary.

Elevation LevelsElevation Levels

Cada nível combina um Effect Style no Figma, um token de sombra (visível no modo light) e uma surface recomendada (mais importante no modo dark). No modo dark, maior elevação = superfície mais clara. O nível 0 é um reset técnico sem Effect Style.Each level combines a Figma Effect Style, a shadow token (visible in light mode), and a recommended surface (more important in dark mode). In dark mode, higher elevation = lighter surface color. Level 0 is a technical reset with no Effect Style.

CSS UtilitiesCSS Utilities

As utilities .ds-elevation-* aplicam apenas box-shadow. Elas existem para casos custom em CSS e não substituem o pareamento entre sombra e surface recomendado nesta página.The .ds-elevation-* utilities apply only box-shadow. They exist for custom CSS cases and do not replace the recommended shadow and surface pairing documented on this page.
