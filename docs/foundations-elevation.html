Elevation

Shadow tokens create visual hierarchy by simulating depth. Higher elevations cast larger, more diffused shadows. These are aligned with the Figma elevation styles.

Focus Ring

The focus ring uses a double-ring pattern for WCAG 2.2 compliance: an inner ring matching the surface color and an outer ring using the primary brand color.

--ds-focus-ring
0 0 0 2px var(--ds-surface-default), 0 0 0 4px var(--ds-color-primary)
