Radio

Radio buttons let users select exactly one option from a group. Built with native <input type="radio"> inside a <fieldset>.

Accessibility

Accessibility Guidelines

Always wrap radio groups in a <fieldset> with a <legend> to provide group context to screen readers. Uses native <input type="radio"> for built-in keyboard navigation (arrow keys move between options within a group). The focus ring is visible on keyboard focus via :focus-visible.
