Toggle

Toggles (switches) let users turn a setting on or off instantly. Built with a styled native checkbox.

Default

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle" role="switch">
  Notifications
</label>

Checked

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle" role="switch" checked>
  Dark mode enabled
</label>

Sizes

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle ds-toggle--sm" role="switch">
  Small
</label>

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle" role="switch">
  Medium (default)
</label>

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle ds-toggle--lg" role="switch">
  Large
</label>

Disabled

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle" role="switch" disabled>
  Disabled off
</label>

<label class="ds-toggle-label">
  <input type="checkbox" class="ds-toggle" role="switch" checked disabled>
  Disabled on
</label>

CSS Classes

ClassDescription
ds-toggleCustom styled switch (native checkbox)
ds-toggle-labelLabel wrapper (wraps toggle + text)
ds-toggle--smSmall size (28x16px)
ds-toggle--lgLarge size (44x24px)

Accessibility

Accessibility Guidelines

Add role="switch" to the checkbox element so screen readers announce it as a switch rather than a checkbox. The native checkbox provides keyboard support (Space to toggle). A visible label should always describe what the toggle controls. Focus ring is visible on keyboard focus via :focus-visible.
