Design System

Menu

Menu é a base visual para listas de comandos. Use Action Menu quando um Button abre uma lista de ações contextuais.Menu is the visual base for command lists. Use Action Menu when a Button opens a contextual list of actions.

Quando usarWhen to use

Use Menu quandoUse Menu when
A interface precisa apresentar comandos, atalhos ou ações de um objeto. Para um trigger completo, use o padrão Action Menu.The interface needs to present commands, shortcuts, or object actions. For a complete trigger pattern, use Action Menu.
Não use Menu quandoDon't use Menu when
O usuário está escolhendo um valor de formulário; nesse caso use Select, Radio ou Checkbox. Se houver busca ou edição de texto, use Combobox.The user is choosing a form value; use Select, Radio, or Checkbox instead. If search or text entry is required, use Combobox.

AnatomiaAnatomy

Item (.ds-menu__item) — comando acionável com label obrigatório.
Ícone (.ds-menu__icon) — opcional, sempre decorativo quando o texto já nomeia a ação.
Shortcut (.ds-menu__shortcut) — opcional, apenas informativo.
Separator (.ds-menu__separator) — separa grupos de ações relacionadas. Item (.ds-menu__item) — actionable command with a required label.
Icon (.ds-menu__icon) — optional and decorative when text already names the action.
Shortcut (.ds-menu__shortcut) — optional and informational only.
Separator (.ds-menu__separator) — separates related action groups.

Action Menu recomendadoRecommended Action Menu

Action Menu combina um Button trigger com um Menu posicionado. O produto deve controlar abertura, fechamento, foco inicial, setas do teclado, Escape e retorno de foco ao trigger.Action Menu combines a Button trigger with a positioned Menu. The product must control open/close behavior, initial focus, arrow keys, Escape, and focus return to the trigger.

<div class="ds-action-menu" data-open="true">
  <button class="ds-button ds-button--outline ds-action-menu__trigger"
          id="project-actions-trigger"
          type="button"
          aria-haspopup="menu"
          aria-expanded="true"
          aria-controls="project-actions-menu">
    <span class="ds-button__label">Project actions</span>
    <i data-lucide="chevron-down" class="ds-button__icon ds-icon"></i>
  </button>
  <div class="ds-menu ds-action-menu__content"
       id="project-actions-menu"
       role="menu"
       aria-labelledby="project-actions-trigger">
    <button class="ds-menu__item" role="menuitem" type="button">...</button>
  </div>
</div>

RegraRuleAplicaçãoApplication
aria-haspopup="menu"No Button trigger do Action Menu.On the Action Menu trigger Button.
aria-expanded + aria-controlsSincronizados com o estado aberto/fechado do popup.Synchronized with the popup open/closed state.
role="menu" + role="menuitem"Use em listas de comandos. Não use para navegação comum ou seleção de formulário.Use for command lists. Do not use for regular navigation or form selection.
