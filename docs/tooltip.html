Tooltip

Small contextual labels that appear relative to a trigger element. The tooltip content is always visible in these demos so you can see each position.

Positions

Tooltip top
Tooltip right
Tooltip bottom
Tooltip left

Usage

Wrap any trigger element with .ds-tooltip and add a .ds-tooltip__content sibling. Use position modifiers .ds-tooltip--top, .ds-tooltip--right, .ds-tooltip--bottom, or .ds-tooltip--left to control placement.
