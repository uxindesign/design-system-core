Form Field

Structured form field wrapper that provides labels, helper text, required indicators, and error messages for form inputs. Use it to create consistent, accessible form layouts.

Accessibility

Always associate labels with inputs using for and id. Link helper text and error messages with aria-describedby. Use aria-required="true" on required inputs and aria-invalid="true" on inputs in an error state.

A field with a label, input, and helper text.

Required field

Add a ds-field__required asterisk to indicate the field is mandatory.

Error state

Apply ds-field--error to the wrapper and add a ds-field__error message. The input should also receive aria-invalid="true".
