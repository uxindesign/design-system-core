Design System

Form Field

Wrapper estruturado de campo de formulário que fornece labels, texto auxiliar, indicadores de obrigatoriedade e mensagens de erro para inputs de formulário. Use para criar layouts de formulário consistentes e acessíveis.Structured form field wrapper that provides labels, helper text, required indicators, and error messages for form inputs. Use it to create consistent, accessible form layouts.

Quando usarWhen to use

Use Form Field quandoUse Form Field when
Envolver qualquer input de formulário que precise de label, texto auxiliar, indicador de obrigatoriedade ou mensagem de erro. Fornece espaçamento consistente e associacoes de acessibilidade.Wrapping any form input that needs a label, helper text, required indicator, or error message. Provides consistent spacing and accessibility associations.
Não use Form Field quandoDon't use Form Field when
O input e independente sem label (raro). Para grupos de Checkbox ou Radio, use <fieldset> com <legend>.The input is standalone without a label (rare). For Checkbox or Radio groups, use a <fieldset> with <legend> instead.

Interação por tecladoKeyboard interaction

Form Field e apenas um container de layout. Não possui interações de teclado próprias. O comportamento do teclado e herdado do componente de input filho (Input, Textarea ou Select).Form Field is a layout container only. It has no keyboard interactions of its own. Keyboard behavior is inherited from the child input component (Input, Textarea, or Select).
