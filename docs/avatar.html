Avatar

Circular user representations in three sizes. Avatars can display an image, text initials, or an icon as a fallback.

When to use

Use avatars when
Representing a user, team, or entity: profile headers, comment authors, team member lists, user menus, and conversation threads.
Don't use avatars when
Displaying a content image (use <img> instead) or creating an icon button (use Button with icon-only variant).

Initials avatar

When no image is available, display 1-2 character initials. Available in three sizes: sm (32px), md (40px), lg (56px).

JD JD JD 
<span class="ds-avatar ds-avatar--sm">JD</span>
<span class="ds-avatar ds-avatar--md">JD</span>
<span class="ds-avatar ds-avatar--lg">JD</span>

Image avatar

When an image is provided it fills the avatar circle. Always include descriptive alt text.

<span class="ds-avatar ds-avatar--md">
  <img src="avatar.jpg" alt="Jane Doe's avatar">
</span>

Icon avatar

Use the ds-avatar--icon modifier with a Material Symbols icon as a generic fallback when neither image nor initials are available.

person person person 
<span class="ds-avatar ds-avatar--md ds-avatar--icon" aria-label="User avatar">
  <span class="material-symbols-outlined">person</span>
</span>

Avatar group

Display multiple avatars together to represent a team or group of users.

AK MR TS +3 
<div style="display:flex;">
  <span class="ds-avatar ds-avatar--md" style="margin-right:-8px;border:2px solid var(--ds-surface-raised);position:relative;z-index:4;">AK</span>
  <span class="ds-avatar ds-avatar--md" style="margin-right:-8px;border:2px solid var(--ds-surface-raised);position:relative;z-index:3;">MR</span>
  <span class="ds-avatar ds-avatar--md" style="margin-right:-8px;border:2px solid var(--ds-surface-raised);position:relative;z-index:2;">TS</span>
  <span class="ds-avatar ds-avatar--md ds-avatar--icon" style="border:2px solid var(--ds-surface-raised);position:relative;z-index:1;" aria-label="3 more users">
    <span>+3</span>
  </span>
</div>

Accessibility

ContextRequirementImplementation
Decorative avatarHidden from assistive technologyaria-hidden="true" or alt="" on image
Meaningful avatar (links to profile)Descriptive alternative textalt="User Name" on image
Icon avatarAccessible name for contextaria-label="User avatar" on container
Initials avatarText content is read by screen readersNo extra attributes needed
ARIA attributes summary
aria-hidden="true" — use on purely decorative avatars (e.g., next to a visible username).
alt="User Name" — use on image avatars that convey identity (e.g., profile links).
aria-label — use on icon avatars to describe the user or context.
