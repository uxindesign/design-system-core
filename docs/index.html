Design System Core

A white-label CSS design system built with 271 design tokens and 20 components. Ships with full light and dark mode support, semantic color aliases, and a complete set of typography, spacing, and radius scales. Drop it into any project with a single stylesheet.

Quick Start

Include the stylesheet in your HTML:

<link rel="stylesheet" href="css/design-system.css">

Dark Mode

Enable dark mode by setting the data-mode attribute on the root element:

<html data-mode="dark">

Light mode is the default. Toggle between modes by switching the attribute value between "light" and "dark".
