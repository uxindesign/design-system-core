Alert

Alerts display important messages to users. Available in filled and subtle styles with optional icons and close buttons.

warning

Accessibility

Accessibility Guidelines

Use role="alert" for important messages that should be announced immediately by screen readers. For non-urgent information, use aria-live="polite" instead. The close button must have an aria-label such as "Dismiss alert". Do not rely on color alone to convey the alert type -- always include an icon and descriptive text.
