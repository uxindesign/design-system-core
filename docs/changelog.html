Changelog

Notable changes to the design system. Follows Keep a Changelog conventions.

v0.5.0 2026-04-15

Documentation overhaul. All 19 component pages rewritten with expanded template. New foundation pages and structural pages added.

added
  • Token Architecture page — 3-layer diagram, alias chain walkthrough, naming convention, cross-format mapping
  • Changelog page (this page)
  • Foundation pages: Motion (7 tokens), Opacity (7 tokens), Z-index (6 tokens)
  • Expanded component template with 10 new sections: when to use, anatomy, best practices (do/don't), content guidelines, token mapping, keyboard interaction, WCAG table, Figma properties, related components
  • All 19 component pages rewritten using the expanded template (+5,000 lines)
  • Do/don't blocks with live component previews across all components
fixed
  • Focus ring implementation in accessibility.html corrected from box-shadow to outline + outline-offset (aligns with ADR-005)
  • Removed incorrect focus ring section from elevation.html
  • Component count on home page corrected to 19
changed
  • Sidebar updated across all 34 pages — added Motion, Opacity, Z-index, Token Architecture, Changelog links
  • Foundations overview: 3 new cards (Motion, Opacity, Z-index) — now 10 total
v0.4.0 2026-04-14

Component token layer and Figma variable alignment.

added
  • Component token JSON files: button.json, input.json, textarea.json, select.json, checkbox.json, radio.json, toggle.json, badge.json, alert.json, card.json, modal.json (118 tokens total)
  • Figma Variables reconciled across 4 collections (Foundation, Brand, Semantic, Component)
  • Style Dictionary build script (build-tokens.mjs) generating CSS in css/tokens/generated/
  • Form Field component (ds-field) with label, helper text, required indicator, error message, character counter
changed
  • Token architecture target formalized: Foundation → Semantic → Component (ADR-001, ADR-005)
  • -default suffix rule enforced: all .default tokens generate -default in CSS
  • Focus ring migrated from box-shadow to outline + outline-offset approach (ADR-005)
  • Brand sublayer formalized: 2 tokens (primary, secondary), no states, swappable per theme
v0.3.0 2026-03

Full component library and documentation site launch.

added
  • 18 components implemented in CSS: Button, Input, Textarea, Select, Checkbox, Radio, Toggle, Badge, Alert, Card, Modal, Tooltip, Tabs, Breadcrumb, Avatar, Divider, Spinner, Skeleton
  • Documentation site with live previews, code blocks, theme switcher, dark mode toggle
  • Foundation pages: Colors, Theme Colors, Typography, Spacing, Radius, Elevation, Borders
  • Guidelines: Theming, Accessibility
  • 3 themes: Default (Blue/Purple), Ocean (Cyan/Indigo), Forest (Emerald/Amber)
  • Light/dark mode with semantic token remapping
v0.2.0 2026-02

Semantic token layer and Figma variables.

added
  • 94 semantic tokens (Theme layer) with Light/Dark mode values
  • Figma Variables: 3 collections (Foundation, Brand, Theme) with mode support
  • 24 text styles: display, heading, body, label, caption, overline, code (Inter + DM Mono)
v0.1.0 2026-01

Initial foundation tokens and project setup.

added
  • Foundation tokens: 10 color palettes (50-950 scale), spacing (20 steps), radius (8 tokens), shadows, opacity, motion, stroke, z-index
  • CSS custom properties (--ds-*) for all foundation tokens
  • Base reset with Inter + DM Mono font loading
  • Repository structure: css/, docs/, js/
