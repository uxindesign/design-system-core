Modal

Dialogs that appear above the page content. Modals block interaction with the rest of the page and require the user to take an action or dismiss.

Accessibility

Use role="dialog" and aria-modal="true" on the modal container. Link the title with aria-labelledby. Trap keyboard focus inside the modal while open and return focus to the trigger element on close.
