Funcionalidades

Tudo que você precisa, pronto para usar

Tokens gerados automaticamente do Figma, componentes testados com axe-core, documentação viva.

300+ Tokens semânticos

Arquitetura de 2 camadas: Foundation e Semantic. Cada valor tem intenção clara e contexto de uso.

19 Componentes

Button, Input, Card, Modal, Alert e mais. Cada um com variantes, estados e documentação de uso.

Dark mode nativo

Adicione data-mode="dark" em qualquer elemento. Tokens semânticos trocam automaticamente.

WCAG 2.2 AA

Contraste validado, focus rings visíveis, ARIA correto. Testado com axe-core em cada release.

Tokens sincronizados via Figma Variables. Mude no Figma, rode um script, CSS atualizado.

CSS puro

Zero dependências de framework. Integra com React, Vue, Svelte ou HTML puro — mesma API.

Como funciona

Em 3 passos

1

Instale

Adicione o design-system.css ao seu projeto. Um único arquivo com tudo compilado.

2

Componha

Use as classes .ds-* no seu HTML. Sem configuração, sem JavaScript obrigatório.

3

Personalize

Troque a paleta brand via data-theme. Tokens semânticos adaptam tudo automaticamente.
